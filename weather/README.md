# Weather Dashboard Assignment Starter

This is a starter monorepo for MPCS 51238 Assignment 4.

## Apps
- `apps/web`: Next.js frontend deployed to Vercel
- `apps/worker`: Node.js worker deployed to Railway

## Core features
- Supabase Auth
- Favorite cities per user
- Background worker polling Open-Meteo
- Supabase Realtime updates on frontend
